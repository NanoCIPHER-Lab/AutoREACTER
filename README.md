# AutoREACTER

A Python-based toolkit for managing and automating reaction modeling in LAMMPS, designed as a component of the **Multiscale Polymer Toolkit (MuPT)**.
This repository is currently in beta and under active development. APIs and functionality may change without notice.

## Quick Start

### Installation
